## Usage

A short example to demonstrate the use of the TikZlings:

```
\documentclass{standalone}

\usepackage{tikzlings}

\begin{document}

\begin{tikzpicture}
\marmot
\end{tikzpicture}	
	
\end{document}
```

An exhaustive list of possible options can be found in [package documentation](./documentation.pdf)
